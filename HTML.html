HTML

HTML, or HyperText Markup Language, was created by Tim Berners-Lee, the creator of the World Wide Web, in 1991 and publicly released in 1993, going through many versions, up to HTML 5.0 in 2014.

HTML is the standart markup language for creating any web page, consisting of multiple elements that tell the browser what to display and how to display it.

Key features of HTML include:

  • Lightweight structure
  • Simple & user-friendly
  • Compatable across multiple devices

There is not much to say, HTML is a very simple and straight-forward language that is easy to pick up and master. With HTML anyone can be a website designer!

And the best part - you don't need a compiler! You can just open up any note app on your computer or phone and just start typing code, at the end save it with the .html extenstion at the end of the file name and you're done!
