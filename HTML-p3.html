HTML Attributes

All HTML elements can have attributes that provide extra information about it. There are many ways to add an attribute:

  • The <a> tag specifies text as a hyperlink, a shortcut to another site, and the href attribute specifies the destination of it.
    Example: <a href="https://www.google.com/">Visit Google.com</a> - Output - Visit Google.com
  • The <img> tag is used to add an image to a page, and the src attribute specifies where the image file is located, and the alt attribute follows it to specify an alternate text for the image if, for some reason, the image cannot be displayed or when you hover the mouse over the image. There should also be width and height attributes that specify the size of the image in pixels.
    Example: <img src="img_cat.png" alt="My Cat">
  • The style attribute is used to add style to an element, it being a color, font, size and more.
    Example: <p style="color:blue;">This text is blue</p> - Output - This text is blue
  • The title attribute defines some extra information about an element when you hover your mouse over it.
    Example: <p title="This is a tooltip">This text has a tooltip (hover over it!)</p> - Output - This text has a tooltip (hover over it!)
  • The lang attribute can be added to the <html> tag to declare the language of the page.
    Example: <html lang="en">

There are also more important attributes later on, like class or id that give certain elements unique identificators, mostly used for styling. class attributes can be used by multiple elements, but id can be used only by one within a page.
