Getting started with HTML

Everything in HTML code is made out of tags, from text to pictures, to styling them, tags are everything. Tags come in a pair, opening and closing (closing tags always have a /), for stability and clarity.

Let's go over how a page is made!

<!DOCTYPE html>
<html>
<head>
<title> Title </title>
</head>
<body>

<h1> Heading goes here </h1>
<p> Paragraph goes here </p>

</body>
</html>


Let's go over the example:

  • The <!DOCTYPE html> tag defines that the document is an HTML5 document. This is one of the few tags that doesn't need a closing tag.
  • The <html> tag is the root element of an HTML page
  • The <head> tag contains meta info about the page, like styles, scripts, charater sets or other things
  • The <title> tag specifies the pages title, which is shown in the browsers title bar
  • The <body> tag defines the document's body, where all the visible media is going to be placed in
  • The <h1> tag is a text element that defines a heading. Heading tags have six sizes, h1 to h6, h1 being the largest and h6 being the smallest
  • The <p> tag is a text element that defines a paragraph
