HTML Classes and CSS

As mentioned previously, you can add classes and IDs to tags to make styling easier. For example, if I want to style multiple boxes the same way in one page or across multiple pages, but I don't want to repeat the same code over and over again, I can just assign the <div> (a container for grouping elements) the class box.

Style sheets can be in the <head> part of the code or it can be in a style.css file. If we want to style an ID, we need to start the style sheet with a hashtag #, but if it's a class, then a period . - #button or .buttons.

Remember: classes can be assigned to multiple elements, ID - only to one!

<style>
  .box {
    width: 200px;
    height: 200px;
    background-color: #c2bcef;
    border: 2px solid rgb(220, 47, 47);
    padding: 20px;
    text-align: center;
    font size: 20px;
  }
</style>

<div class="box">
  This is a box.
<div>

Output:

This is a box.
