C++ Loops

Loops can execute a block of code as long as a condition has been reached. Loops are useful because they help save time, reduce erros and make code more readable.

There are two types of loops - for loops and while loops.

While loops go thought a block of code as long as a condition is true.

For loops are used when you know exactly how many imes you want to go thought a block of text.

You can also place a loop inside a loop, making a nested loop.


        while (condition) {
        // code block to be executed
        }
              

        for (statement 1; statement 2; statement 3) {
        // code block to be executed
         }
