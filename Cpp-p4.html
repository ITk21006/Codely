C++ User Inputs

In the first exercise we already talked about outputting text with cout but now we will also learn about cin.

cin (pronounced "see-in") is a command that reads data that is inputed from the keyboard, and it uses the extraction operator (>>), for example,
int age;
cout << "What is your age?";
cin >> age;
can be used to ask the user to input their age, using the word "age" itself as an int.

It can be used to input any form of variables, and how you use it is only limited by your imagination. For now, we will use it to make a simple calculator.

                
                  #include <iostream>
                  using namespace std;
                  
                  int main() {
                    double x, y;
                    double sum;
                    cout << "Type the first number (x): ";
                    
                    cout << "Type the second number (y): ";
                    
                    sum = x + y;
                    cout << "Sum is: " << sum;
                    return 0;
                  }
