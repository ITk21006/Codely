C++ Variables

Next on the list - variables.

Variables are a sort of container for a data value, and those value types can be a number, a single word or a string of text or even a true/false statement.

For each variable type there is a different keyword:

  • int - stores integers or whole numbers (111 or -111)
  • double - stores floating point numbers (1,23 or -12,3)
  • char - stores a single charater like 'X' or 'a' (single quotes needed)
  • string - stores a line of text like "Hello world!" (double quotes needed)
  • bool - true or false value

To create a variable, all you need to do is specify what type it is and what's it's value, as well as give it a name or identifier -


type variableName = value;
