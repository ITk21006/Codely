HTML Text Formatting

Apart from using style to change how the text looks, you can add formatting tags within the text itself to quickly add some style spark!

For text formatting we can use:

  • <b> and <strong> tags to make the text bold;
  • <i> tag to make the text italic;
  • <em> tag to emphasize the text;
  • <mark> tag to mark the text;
  • <small> tag to make the text smaller;
  • <del> tag to make the text look deleted;
  • <ins> tag for inserted text;
  • <sup> tag for superscript text;
  • <sub> tag for subscript text.

And that's it! All you need to do to implement these is to put text between the start and end tags of the style, like this:
<p>This text is normal, while <b>this text is bold</b></p> - Output - This text is normal, while this text is bold
